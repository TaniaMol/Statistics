# Gender and the environment in Mexico

In order to analyze my data, the following programs were written:

- **MoCT.py**
  - Measures of Central Tendency
  - Normal distribution graph
  - Sampling distribution
  - z-table

- **DepT-test**
  - Two tailed T-test
  - Differences of means graph
