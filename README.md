# MoCT.py
Retrieving, Processing, and Visualizing Data with Python

To use this for your own data, you'll need to install these packages:

- <a href="https://docs.python.org/2/library/math.html" rel=noopener>math</a>
- <a href="https://pypi.python.org/pypi/numpy" rel=noopener>numpy</a>
- <a href="https://pypi.python.org/pypi/pandas/0.20.3" rel=noopener>pandas</a>
- <a href="https://pypi.python.org/pypi/scipy/0.19.1" rel=noopener>scipy</a>
- <a href="https://pypi.python.org/pypi/matplotlib/2.0.2" rel=noopener>matplotlib</a>

Then make sure that the data you want to analyze is in the same folder of the script.
When you run the program, you'll see a set of easy instructions to follow.
You'll get:

- **N**
- **Mean**
- **Sum of squares**
- **Variance**
- **Population Standard Deviation**
- **Standard Error**
- **Graph for population distribution**

Then you can choose sample means and values to see how far they are from the mean... and get:

- **Standard Error for each sample**
- **Z-score**
- **Z-table value for that score**
- **Probability of getting that value**
- **Graph for sample distribution**

You can download **scores.txt** to test the script.
